## 背景デザインの呼応 (キャラ ⇔ 背景)

- **金髪 + 暖光の表情** ⇔ **暖色木目床 + 窓からの朝光** = 暖色基調の統一
- **青の瞳 + ナビーtrack jacket + ナビースクランチー + ナビーsports bra** ⇔ **紺のロッカー + 紺アクセントの壁帯 + 青タオル + 水色ヨガボール** = **ナビーアクセントの三重以上の繰り返し** (キャラと空間の色彩同調)
- **白いタオル (右肩)** ⇔ **白い壁・天井・トレーニングベンチの白フレーム** = 純白の継承
- **プロテインシェイカー (ピンク液 + 紺蓋)** ⇔ **ジム機材 (筋トレの結果と原因)** = 装具とジム空間の文脈整合 (運動 → 補給)
- **toned abs (運動した体)** ⇔ **ダンベル+ケーブルマシン+ランニングマシン** = 「彼女がたった今使った機材」の暗示
- **post-workout の汗+紅潮** ⇔ **ジム室内の朝光+運動後の atmosphere** = 「ちょうど運動が終わった瞬間」の空間化
- **playful な微笑み** ⇔ **明るいジムの清潔感** = 親しみやすさの整合
- **3-stripe のアスレチックパターン** ⇔ **窓フレームの縦ライン + ロッカーの縦ライン** = 縦線リズムの呼応

PLR ROSTER 中で最も「日常+健康+親しみやすさ」が密度高く組まれたペア。OPP 系の煌びやかな運動系 (アイドル/姫君/精霊) と異なり、**最も身近な日常感**が突出。PLR07 カイ (空の冒険者・登山系) や PLR04 蓮 (剣道部・武道系) と並ぶ「フィジカル系」三項対立: **冒険・武道・フィットネス**。
