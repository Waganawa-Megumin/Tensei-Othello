## 📝 検証チェック (各キャラ受領時)

各キャラを受け取ったら以下を自動チェック:

- [ ] 解像度 1024×1024 (違えばリサイズ)
- [ ] 背景が透明 (白の場合は自動透過処理)
- [ ] 円形フレームが焼き込まれていない
- [ ] キャラクターの維持 (顔・髪・服装・小物)
- [ ] 顔の中心が画像中心 (±5%)
- [ ] 頭頂部が切れていない
- [ ] クオリティアップが見られる

## 🛠️ Claude 側の自動処理フロー

各キャラ受領時に Claude が実施:

1. **検証** (上記チェック項目)
2. **正規化**:
   - 必要なら 1024×1024 にリサイズ
   - 背景が白なら透過処理
   - ファイル名: `{ID}_{slug}.png` (例: `PLR00_default.png`)
3. **保存**: `processed/players/` または `processed/opponents/` に格納
4. **トラッカー更新**: このファイルを ✅ に更新
5. **次のキャラへ案内**

## 📦 完成後の納品 (自動処理)

41 体すべて完了したら:

1. `processed/players/` (21枚) と `processed/opponents/` (20枚) を整理
2. 既存ゲーム公開ディレクトリ構造に合わせて配置: `public/avatars/players/` `public/avatars/opponents/`
3. Claude Code への差分ハンドオフ ZIP を作成 (画像 + 配置指示)
4. サイズが 30MB を超えれば pack 分割
5. 必要なら `manifest.json` 更新 (バージョン番号や描画順)
