## ① NotebookLM ソース文書

```markdown
# 召喚されたらオセロ世界でした！ — ゴールデンウィーク開発記録

## 概要
**スマホ1台**と AI ツールだけで、ゴールデンウィーク(2026年5月)に作り上げた異世界召喚オセロ RPG。PC は使わず、**電車や車での移動中・山や自宅で寝ころびながら・ちょっとした隙間時間**に、スマホから AI に指示して開発した。React 製の Web アプリ(PWA、オフライン対応)。「異界〈盤上世界〉へ召喚され、20人の達人を順に打ち破る」物語モードを持つ。

## 制作スタイル（このストーリーの心臓）
- 道具は**スマホ1台**のみ。PC なし。
- 作った場所: **電車の中・車での移動時間・山・自宅のベッドで寝ころびながら**。
- 作った時間: 連休の**隙間時間**。まとまった机の前の時間ではない。
- やったこと: 指で AI に指示 → Claude Code が実装 → GitHub に積み上がる。「寝ころびながら RPG が育っていく」体験。

## ヘッドライン数字
- 道具は **スマホ1台**（PC なし）／ 場所は 電車・車・山・布団の上
- 295 コミット / 28 日間
- うち 84%（249コミット）が GW 6日間(5/2–5/7)に集中、平均 41.5 コミット/日
- キャラクター 43 体（プレイヤー 21 + 対戦相手 22）
- イラスト 84 枚
- TypeScript 25,547 行
- 対応言語 2（日本語・英語）
- 物語 20 章、真エンディングまで最大 約421 戦
- AI 8 段階（乱択 → 貪欲 → 定石評価 → ミニマックス深さ1〜4）
- 第三者観点のセキュリティスキャンを実施（ChatGPT Codex）
- ライブ公開中（PWA・オフライン可）

## AI 開発ワークフロー（本作の主役）
- 司令塔: 開発者本人 — 物語・世界観・戦略・QA
- Claude アプリ Chat/Code: 設計相談 + 実装。Claude Code を複数セッション(3ブランチ)並走、`claude/<task>` ブランチ → main へ統合
- GitHub: リポジトリ + GitHub Actions 3本（型チェック+テスト / GitHub Pages 自動デプロイ / Cloudflare Workers デプロイ）
- Cloudflare Workers: Anthropic API へのプロキシ。棋譜レビュー = Claude Sonnet 4.6（構造化レビュー）、実況コメント = Claude Haiku 4.5
- ChatGPT: 全キャラ・全シーンのイラスト生成（43体 + 84枚）
- ChatGPT Codex: 第三者観点でのコードのセキュリティスキャン（クラウド上で脆弱性を検査し、指摘を確認・対応）
- 公開: GitHub Pages（PWA、Service Worker 自動更新）
- データフロー: スマホ → Claude Code(3セッション) → GitHub(CI/CD) → ①GitHub Pages(PWA) / ②Cloudflare Workers → Anthropic API。絵は ChatGPT、コードのセキュリティ検査は ChatGPT Codex。

## 技術スタック
React 18.3 / Vite 5.3 / Tailwind CSS 3.4 / TypeScript 5.5 / lucide-react / vite-plugin-pwa 1.2 / Vitest / Cloudflare Workers(Wrangler) / Anthropic API(Sonnet 4.6 + Haiku 4.5) / GitHub Actions / GitHub Pages / ChatGPT Codex(セキュリティスキャン)

## 成果物スケール
- 物語モード: 20章 × プレイヤー21体の連鎖アンロック。真エンディングまで最大 約421戦
- フリーモード / 2人対戦
- AI 8段階: Lv1-2 乱択, Lv3-5 貪欲, Lv6-9 定石評価, Lv10-14 ミニマックス深さ1-2+αβ枝刈り, Lv15-17 深さ3, Lv18-20 深さ4+機動力評価
- キャラ43体: プレイヤー21（隠し英霊ハルキ含む）+ 対戦相手22（隠し「ゼロ現世帰還」「ヴォイドφ」含む）
- 真エンディング + 隠しキャラ解放
- 棋譜(きふ)保存/読込, ヒント, 待った, 10セーブスロット, シーン回想
- 物語テキスト総量: シナリオ資料 約4,486行 + 多言語ストーリー 約2,254行
- PWA: オフライン対応, Service Worker 自動更新

## ゴールデンウィークのタイムライン（日別コミット）
- 5/2: 55（始動 — 基盤 + 20体ロスター）
- 5/3: 30 / 5/4: 28（物語フレーム・セーブ・AI段階）
- 5/5: 7（整理）
- 5/6: 53 / 5/7: 56（ピーク — 全物語ローカライズ・43体アバター・真エンディング5段シネマ）
```

## 使い方（NotebookLM）

1. NotebookLM で新規ノートブックを作成
2. **ソース**として ① を貼り付け（または本ファイルをアップロード）
3. Studio → **Infographic** を選択
4. カスタムプロンプト欄に ② を貼り付け
5. 生成 → 縦長で確認。数字・配色・連携図の粒度は ② を微調整して再生成

## 留意点

- 「スマホ1台・移動中/寝ころび」の制作スタイルは事実として前面に。情景カット(電車/車/山/布団)の枚数や順序は好みで調整可。
- ChatGPT Codex によるセキュリティスキャンは「実施・対応」止まりの記載。検出件数や重大度を載せたい場合はスキャン結果を確認して追記。
- 数字(コミット/キャラ/行数等)は実測。生成画像の表示が ① と一致するか目視確認。
