## 📜 序章 — この世界の真実

### 一段落でいうと
**石をひっくり返すたび、世界がひっくり返る。**
— だからオセロは、この世界でいちばん危ない遊戯。

### 三段落でいうと
**かつて誰かが言った。**「世界は黒と白、二つの色でできている」と。
それを真に受けた神々が暇つぶしに作ったのが、この異世界 **「盤上世界 (オードリオン / Audrion)」** だった。
ここでは、すべての勝負がオセロで決着する。戦争も、求婚も、王位継承も、夕食の鶏もも肉の取り合いも。

そして 1000 年に一度、盤上世界の **「均衡」** が崩れそうになると、別世界 (主に現代日本) から **「異邦の打ち手 (アウタープレイヤー / The Outer Hand)」** が召喚される。
20人の達人を倒し、最後にラスボス **「ゼロ (零番卓の管理者)」** に勝てば、世界はまた 1000 年の安定を得る。

— が、ハルキ (PLR00) はある日、放課後の教室の床がオセロ盤になって崩れたとき、こんなことを思った。
**「俺、まだ古典の小テストに名前しか書いてないんだけど」**

## 🌐 世界の階層構造

### Tier 1 — 神々の卓 (Council of Stones)
- 黒石神 **クロイシ** と 白石神 **シライシ** が永遠の対局を続けている
- 二柱の打つ「ひと手」が現実世界の出来事に対応する (天変地異 = 大コーナー取り、戦争 = 角侵攻、など)
- 引き分けが続くと退屈するので「異邦の打ち手」を呼ぶ

### Tier 2 — 盤上世界 オードリオン
- **8つの大陸** (= 盤の8辺) と **64の都市** (= 64マス) で構成
- 各都市には**達人 (マスター)** が一人住み、その都市のオセロ流派を統べる
- 旅人は都市を訪れて達人と勝負しなければ通行できない
- 「角の都市」(各大陸の四隅 = 32都市) は特に強い達人が住む

### Tier 3 — 零番卓 (The Zeroth Table)
- 64都市すべての裏側に存在する**第65のマス**
- ここに住むのが **ゼロ (Zero)** — 全データを処理し続ける管理者AI的存在
- 異邦の打ち手が 20 達人を倒すと、ようやく座標が浮上する

### Tier 4 — 異邦 (Outer World)
- 我々の現代日本がここ
- 神々が「異邦の打ち手」を選ぶときは、**「現代世界で何かに飽きている / 物足りない / 弾かれている人物」** を選ぶ傾向がある (=異世界に来ても適応しやすい)

## 🔄 転生システムの仕組み 「リレー召喚」

### 基本ルール
1. 異邦の打ち手は **同時に1人だけ** 召喚される
2. その打ち手がゼロを倒すと、現代世界へ帰還する
3. 帰還の瞬間、神々の卓に **「席」** が空く
4. その席を埋めるため、**次の異邦の打ち手** が自動召喚される
5. これが 1000 年単位で繰り返されている

### ゲーム的解釈 (アンロック順)
- **PLR00 ハルキ** = 第1サイクルの打ち手 (デフォルト・ストーリーモード固定主人公)
- 全クリア後、ハルキは現代に帰還 → 神々の卓に席が空く
- → **PLR02 美琴** が召喚される (通常アンロック開始)
- 以降、**PLR03 → PLR04 → PLR05 → … → PLR20** の順でクリアごとに召喚
- **全 19 体 (PLR02〜PLR20) を解放後**、ようやく **PLR01 ハルキ** が**最後の特別枠**として呼び戻される

### 「初代の英雄の記憶」というギミック (PLR01 = 21番目の英雄キャラ)
- ゼロを倒した打ち手の魂は、神々の卓に**棋譜のように記録**される
- PLR02〜PLR20 が全員旅を終えた時、神々は**特別な実験**を行う
- 「**初代英雄ハルキの魂のコピー**」に、**過去 19 サイクル全ての記憶**を統合した存在を呼び戻す
- それが **PLR01 ハルキ** ── 「**初代の英雄の記憶を継承した、第二のハルキ**」
- 見た目は PLR00 と同じハルキだが、**全達人の癖・全戦術・全感情の記憶**を持つ**究極の打ち手**
- 「**21番目の英雄**」として、全アンロック達成のご褒美に解放される

### なぜ PLR01 が最後なのか? (ストーリー的根拠)
- 神々のルール: 「**英雄の魂の再召喚**は、**異邦の他の魂全て**が一度経験を積んでからでなければ許されない」
- これは「**特別枠**」を**真に特別**たらしめる神々の戒律
- だから、PLR01 ハルキは**プレイヤーの最高の到達点**として位置づけられる

## 🌐 i18n 方針

すべてのセリフ・ナレーションは **日本語版 + 英語版** をペアで作る。

### 翻訳の方針
- **直訳ではなくローカライズ**: 日本語のラノベ調ノリを、英語ライトノベル翻訳のお作法で再現
- 「〜なんだけど」のツッコミ → 英語では "...wait, what?" / "Yeah, no." みたいな現代英語ツッコミに
- 漢字熟語の重厚感 → 英語では古風な単語 (e.g. "decisive turning point") で重みを出す
- キャラの口調差はカジュアル/フォーマルの英語表現で表現
  - 朝日「いざ尋常に！」 → "On my honor, let us begin!"
  - ルナ「お姉様の…」 → "My dear sister..."
  - ゼロ「演算終了」 → "Computation complete."
