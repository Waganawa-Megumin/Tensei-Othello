## 背景デザインの呼応 (キャラ ⇔ 背景)

- **黒髪 + 紫undertones** ⇔ **暗い大聖堂内部 + 紫のベルベットドレープ** = 暗紫色調の統一
- **クリムゾン赤の瞳** ⇔ **赤いバラ + 赤い蝋燭の炎** = 主調差し色 (血/薔薇/炎の赤) の繰り返し
- **牙 (キャラ)** ⇔ **月の前のコウモリのシルエット (背景)** = **vampire 記号の二重鏡像** (キャラの牙と背景のコウモリ群、同じ「吸血鬼」シンボルが二箇所に)
- **シルバーのコウモリチャーム (キャラ)** ⇔ **飛ぶコウモリ・月 (背景)** = 三重の bat motif (装具+モチーフ+月のシルエット)
- **ルビーブローチ** ⇔ **赤いバラ + 蝋燭の炎** = 紅の宝石性の呼応
- **gothic 衣装の paisley/damask 柄** ⇔ **gothic アーチ建築・装飾柱** = 同じゴシック美学
- **黒の襟+赤のライニング** ⇔ **黒い暗闇+赤いバラの蔓** = 黒赤の対比繰り返し
- **静かな自信の表情** ⇔ **月の下の永遠の夜** = vampire の「不死」を空間化
- **舞い散る赤いバラの花弁** ⇔ キャラの「血と薔薇の館の主人」感の visualizing

PLR ROSTER 中で最も「ダーク・ゴシック・神秘」が高密度に重なるペア。OPP06 つむぎ (獣使い) と動物使いつながりで対比可能だが、PLR17 = 不死の貴族 / OPP06 = 自然と共生する獣使い、で性質が真逆。
