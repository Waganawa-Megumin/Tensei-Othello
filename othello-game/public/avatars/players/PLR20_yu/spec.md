## 背景の世界観

**神域のプラザ — 雲上の白大理石神殿** (Divine plaza — marble pavilion above the clouds)

`background.png` (1024×1024 RGB) として同梱:

- 中央上空に**強い太陽光柱** (神々の光、放射状に広がる)
- 上空の遠景に**雲海に浮かぶ神殿/城** (黄金のドーム、新古典主義建築 = ギリシャ・ローマ風)
- 中景・遠景に**雪をかぶった山々** + 雲海

## 背景デザインの呼応 (キャラ ⇔ 背景)

- **銀/プラチナ髪** ⇔ **白大理石柱・白雲** = 純白統一の主調色
- **青い瞳 + 青の髪飾り** ⇔ **青空 + 青葉のアクセント + 雲の影の青** = 冷色アクセント呼応
- **金色の蝶々の髪飾り** ⇔ **柱の金装飾 + 金の蔓 + 金の儀式壷** = 金の暖色装具呼応
- **着物の青と白の花柄** ⇔ **青葉 + 白い花の前景** = 自然モチーフの繰り返し
- **金縁の扇子** ⇔ **金のマンダラ床面 + 金の柱装飾** = 儀礼性の継承
- **穏やかで知的な微笑み** ⇔ **神聖な光柱 + 雲上の神殿** = 神性の整合
- **静かな立ち姿** ⇔ **円形マンダラ床面の中央** = 構図の中央配置

PLR ROSTER 中で最も「品格・神域・静謐」が頂点に達するペア。PLR12 エル (森の精霊・暖光の神域) との対比軸: **PLR12 = 自然の神域 / PLR20 = 神話の神域** (両方とも「神性」だが、PLR12 = 動的な森の生命 / PLR20 = 静的な天界の品格)。
