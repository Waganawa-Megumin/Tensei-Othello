# avatars/ — キャラクター素材パッケージ

「召喚されたらオセロ世界でした！」(Summoned as an Othello Player) のキャラクター素材。

## 構造

```
avatars/
├── README.md         このファイル
├── opponents/        対戦相手 20 体
│   ├── INDEX.md      20 体一覧
│   ├── OPP01_ichika/
│   │   ├── character.png    1024×1024 RGBA
│   │   ├── background.png   1024×1024 RGB
│   │   ├── icon.png         1024×1024 RGBA (円形)
│   │   └── spec.md          このキャラの仕様書
│   └── ... (OPP20_zero まで)
└── players/          プレイヤー 21 体 (PLR 完成後に同構造で追加予定)
```

## 設計方針

1. **フォルダ名 = 識別子**: `OPP01_ichika/` のようにフォルダ名にキャラ ID を含める。
2. **フォルダ内ファイル名は汎用**: `character.png` 等の決まった名前で統一。Claude Code 側からは `${id}_${slug}/character.png` でアクセスしやすい。
3. **仕様書を各キャラに同梱**: 単独で配布・再生成できるよう、各フォルダが自己完結。
4. **3 種の画像レイヤー**:
   - `character.png` (透過済キャラ単体)
   - `background.png` (背景単体)
   - `icon.png` (上記 2 つを合成し円形クロップ済)
5. **解像度は全て 1024×1024** (前世代の 256×256 簡易版から大幅アップ)。
6. **透過処理は Adobe Photoshop で外部処理** (このパッケージは外部処理済みの最終納品物)。
